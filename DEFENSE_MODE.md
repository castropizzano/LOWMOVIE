# Defense Mode

Documentação do modo de apresentação para a defesa da dissertação.

## O que é

O Defense Mode (`/defense`) é uma interface otimizada para projeção durante a defesa da dissertação perante a banca examinadora. Permite navegar pela pesquisa de forma sequencial e controlada, com foco na apresentação oral.

## Acesso

```
https://lowmovie.lovable.app/defense
```

## Navegação

| Ação | Controle |
|---|---|
| Próximo bloco | `→` ou `↓` ou `Espaço` ou clique |
| Bloco anterior | `←` ou `↑` |
| Início | `Home` |
| Final | `End` |
| Sair | `Esc` |

## Recomendações para apresentação

1. **Resolução:** Usar monitor/projetor em 1920×1080
2. **Navegador:** Chrome ou Firefox em tela cheia (`F11`)
3. **Conexão:** Testar carregamento das imagens antes da defesa
4. **Backup:** Ter a dissertação em PDF aberta como alternativa

## Complementos

Durante a defesa, o portal pode ser usado em conjunto com:

- **Lowbyrinth™** (`/mapa`) — para demonstrar a navegação não-linear
- **Mapa Conceitual** (`/mapa`) — para visualizar conexões teóricas
- **Seções individuais** — para aprofundar pontos específicos levantados pela banca
